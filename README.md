# garbage-classification-project
Ứng dụng AI phân loại 12 loại rác thải từ hình ảnh với giao diện hiện đại và độ chính xác cao.

# Phân Loại Rác Thải Bằng AI

## Cài đặt
```bash
pip install -r requirements.txt

Sử dụng
Đặt các file model .h5 vào thư mục models/

Chạy: python app.py

Cấu trúc
models/ - Chứa file .h5 (có sẵn)

saves/ - Lưu kết quả và lịch sử

core/ - Xử lý model và lịch sử

ui/ - Giao diện

utils/ - Tiện ích
